How this simulated voting works?

  • Each Vote is considered as a block.
  • Each block will have a unique hash generated and is linked to it.
  • Next block will only be valid if the previous block's hash matches with the current block.
  • Similar blocks will be added continuously developing a chain of blocks called Blockchain.
  • If someone tampers with the registered votes, the corresponding block's hash will be changed which in turn will not match with the previous and next block.
  • This causes the entire Blockchain to get broken.
